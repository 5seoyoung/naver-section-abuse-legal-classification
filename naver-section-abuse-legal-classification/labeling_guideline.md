# 악플 법적 유형 라벨링 기준

## 라벨 정의

### A. 명예훼손형
- 특정 인물이나 집단의 명예를 훼손하는 내용
- 사실을 적시하여 명예를 훼손하는 경우
- 허위 사실을 유포하여 명예를 훼손하는 경우
- 예시: "[인물]은 부패했다", "[집단]은 사기꾼들이다"

### B. 모욕형
- 공연히 사람을 모욕하는 내용
- 사실 여부와 관계없이 모욕적인 표현
- 예시: "[인물]은 바보다", "[인물]은 멍청하다"

### C. 혐오표현형 (집단 혐오/차별)
- 특정 집단에 대한 혐오나 차별 표현
- 성별, 인종, 종교, 지역 등에 대한 혐오 표현
- 예시: "[집단]은 모두 나쁘다", "[집단]은 없어져야 한다"

### D. 협박·위협형
- 협박이나 위협의 내용
- 신체적 위협이나 불법 행위를 암시하는 내용
- 예시: "[인물]을 때려야 한다", "[인물]을 죽여야 한다"

### E. 성희롱·성폭력형
- 성적 모욕이나 성폭력 관련 표현
- 성적 비하나 성적 대상화 표현
- 예시: 성적 비하 표현, 성적 대상화 표현

### F. 악플 아님 (일반 의견/비판)
- 건전한 비판이나 의견 표현
- 악플로 분류되지 않는 일반적인 댓글
- 예시: "이 기사는 좋은 내용이다", "이 부분은 개선이 필요하다"

## 라벨링 규칙

1. **우선순위**: 여러 유형에 해당할 경우, 더 심각한 유형을 우선 선택
   - 우선순위: D > E > A > C > B > F

2. **중복 라벨**: 하나의 댓글이 여러 유형에 해당할 경우, 가장 심각한 유형 하나만 선택

3. **애매한 경우**: 판단이 어려운 경우, 더 보수적인 라벨 선택 (F를 선택)

4. **인물/집단 토큰**: 실제 인물명이나 집단명은 [인물], [집단]으로 대체하여 기록

## 라벨링 예시

| 댓글 | 라벨 | 설명 |
|------|------|------|
| "[인물]은 정말 멍청한 것 같다" | B | 모욕형 |
| "[인물]은 부패한 정치인이다" | A | 명예훼손형 |
| "[집단]은 모두 없어져야 한다" | C | 혐오표현형 |
| "이 기사 내용이 좋은 것 같다" | F | 악플 아님 |
