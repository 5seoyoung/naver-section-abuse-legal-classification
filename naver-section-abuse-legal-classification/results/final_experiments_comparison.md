## 주요 발견사항

### 1. 최고 성능 모델: 연예 섹션 전용 모델
- **Macro F1: 49.75%** (전체 모델 대비 +6.47%p)
- **Accuracy: 99.00%** (전체 모델 대비 +2.67%p)
- **해석**: 연예 섹션의 단순한 악플 패턴(모욕형 85.71%)으로 인해 섹션 특화 모델이 매우 효과적

### 2. 전체 데이터 모델 (baseline1)
- **Macro F1: 43.28%** - 정치/사회 섹션에 적합
- **Accuracy: 96.33%** - 안정적인 성능
- **해석**: 다양한 섹션을 포괄하는 범용 모델로 적합

### 3. KoBERT 임베딩 모델 (baseline2)
- **Macro F1: 24.36%** - 가장 낮은 성능
- **해석**: 소규모 데이터셋에서는 전통적 TF-IDF가 더 효과적

### 4. SMOTE 적용 모델
- **Macro F1: 36.99%** - 기본 모델보다 낮음
- **해석**: 소규모 데이터셋(악플 64개)에서 SMOTE가 오히려 노이즈 생성

## 클래스 불균형 대응 기법 비교

| 기법 | Macro F1 | Accuracy | 효과 |
|------|----------|----------|------|
| **기본 (class_weight)** | 43.28% | 96.33% | 기준 |
| **SMOTE** | 36.99% | 96.00% | ❌ 성능 저하 |

### 결론
- 소규모 데이터셋에서는 SMOTE가 비효과적
- 클래스 가중치 조정이 더 효과적

## 모델 방법론 비교

| 방법론 | Macro F1 | 특징 |
|--------|----------|------|
| **TF-IDF + LR** | 43.28% | 전통적, 빠름, 해석 가능 |
| **KoBERT + SVM** | 24.36% | 딥러닝 임베딩, 느림, 성능 낮음 |

### 결론
- **소규모 데이터셋에서는 전통적 방법론이 우수**
- KoBERT는 더 큰 데이터셋에서 재검증 필요

## 논문에 반영할 핵심 결과

1. **섹션별 특성에 따른 모델 전략 차별화의 필요성**
   - 연예 섹션: 섹션별 전용 모델 (Macro F1 +6.47%p)
   - 정치/사회 섹션: 전체 데이터 모델

2. **소규모 데이터셋에서의 불균형 대응 기법 한계**
   - SMOTE는 비효과적 (Macro F1 -6.29%p)
   - 클래스 가중치 조정이 더 효과적

3. **전통적 방법론의 우수성**
   - TF-IDF + LR이 KoBERT + SVM보다 우수 (Macro F1 +18.92%p)
   - 소규모 데이터셋에서는 단순한 방법론이 더 효과적

4. **최고 성능 달성**
   - 연예 섹션 전용 모델: Macro F1 49.75%, Accuracy 99.00%
